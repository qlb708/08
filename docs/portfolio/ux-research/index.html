用研方法知识库
访谈法（深度访谈）
适用场景：探索用户动机、痛点、决策路径。
关键技巧：用「为什么」追问，避免封闭式问题，单次访谈30-60分钟，样本5-8人即可得到饱和信息。
输出：用户洞察报告、用户故事。
可用性测试
适用场景：验证设计方案、发现交互问题。
方法：让用户在真实环境完成任务，观察而非引导。5个用户能发现85%的可用性问题（Nielsen定律）。
输出：问题清单 + 优先级矩阵。
NPS + 问卷调研
适用场景：量化用户满意度、大样本验证假设。
NPS基准：60+为优秀，40-60良好，<40需关注。
问卷设计原则：单题只问一个维度，选项互斥穷举，总题数≤15道。
