常见问题

关于这些工具，你可能想知道的。

这些 Demo 是真实可用的还是演示性质的？
全部真实可用。AI 工具类 Demo 调用 Google Gemini API，数据实时生成，不是预设好的样本数据。竞品雷达的数据来自真实的社交平台热点采集。
用了哪些技术？
前端使用 Vue 3 + Vite（Vibe-coding 方式构建），部署在 GitHub Pages。AI 调用使用 Google Gemini 1.5 Flash。数据存储在 GitHub 仓库，通过 GitHub API 动态更新。整个项目从构思到上线不超过两天。
AI 生成的内容质量怎么样？
每个工具都有精心设计的 Prompt，输出内容有固定的专业结构框架（如用研报告的 7 个章节），质量稳定可用，可以作为真实工作的初稿直接使用。
这些工具体现了哪些产品能力？
体现了 AI 产品设计能力（对话式交互设计）、Prompt 工程能力（专业输出结构设计）、数据架构能力（实时数据管道设计）、以及快速落地能力（Vibe-coding 方式构建 MVP）。
可以在实际工作中使用这些工具吗？
可以。产品文档速写台和用研报告生成器已经在实际工作中用于提升文档效率。竞品雷达也已用于 AI 旅行赛道的日常监测。
