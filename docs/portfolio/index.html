关于这个工具集
和 ChatGPT / Kimi 相比有什么优势？
通用 AI 需要你自己写 Prompt、组织结构，每次都要重新解释背景。这里的工具把产品运营场景的 Prompt 工程做到了最优，输入框告诉你填什么，输出是行业标准格式，生成即可用。
这些工具是真实在用的吗？
每个工具来自真实工作场景：需求评审来自 PRD 评审前的漏洞排查；人群筛选来自每周处理后台名单的低效；知识库 AI 问答来自看完竞品文档后不知道问谁的场景。
能体现什么产品能力？
三层：(1) 需求挖掘——从真实工作痛点识别高频低效场景；(2) 产品设计——用户路径、信息架构、交互细节独立完成；(3) AI 产品能力——Prompt 工程、多模型降级、AI 输出结构化设计。
