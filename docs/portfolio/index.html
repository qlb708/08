关于这个工具集
和 ChatGPT / Kimi 相比有什么优势？
通用 AI 需要自己写 Prompt，每次重新解释背景。这里的工具把产品运营场景的 Prompt 工程做到最优，输出是行业标准格式（PRD/OKR/用研报告），生成即可用。
这些工具是真实在用的吗？
每个工具来自真实工作场景：需求评审来自 PRD 评审前的漏洞排查；人群筛选来自每周处理后台名单；知识库问答来自看完竞品文档后不知道问谁的场景。
能体现什么产品能力？
三层：需求挖掘（从工作痛点识别场景）、产品设计（用户路径、信息架构、交互细节）、AI 产品能力（Prompt 工程、多模型降级、输出结构化设计）。
